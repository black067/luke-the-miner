# 星环矿工 / Luke the Miner

> 一款复古风格的 HTML 文字/经营游戏，采用 Windows 98 美学。

## 🎮 简介

你是一名被 AI 自动分配去星环矿场还学贷的矿工。在星环上挖矿、战斗、交易、升级装备，努力在债务的阴影下生存下去。

## 🛠 技术栈

- **语言**：TypeScript
- **构建**：`tsc`（TypeScript Compiler）
- **测试**：Vitest
- **样式**：98.css + 自定义 CSS
- **音效**：Web Audio API 合成
